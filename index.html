SOFTWARES

Software é um termo técnico que foi traduzido para a língua portuguesa como suporte lógico e trata-se de uma sequência de instruções a serem seguidas e/ou executadas, na manipulação, redirecionamento ou modificação de um dado ou acontecimento.

Software de aplicativo : Contém todos os programas derivados de uma programação de software e que cumprem uma tarefa específica, em quase todas as áreas da vida cotidiana. São utilizados por dispositivos móveis e computadores.

Software de programação: Por meio do conhecimento lógico e da linguagem de programação orientada a objetos, é possível projetar utilidades digitais para executar várias funções, esses programas são a base na qual o código é escrito para desenvolver novos sistemas dentro de um sistema operacional.

Software de sistema: Entre os tipos de software, o de sistema é o mais importante. É ele que permite ao usuário usar a interface do sistema operacional incorporada ao dispositivo. O software de sistema é composto por um conjunto de programas ou aplicativos nativos, que têm dois propósitos:

1- Gerenciar recursos físicos do dispositivo para coordenar tarefas e administrar a memória para seu uso;

2- Oferecer uma experiência de uso para que seja possível controlar e interagir com o sistema.
