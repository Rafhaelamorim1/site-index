Linux

Linux é um sistema operacional assim como Windows e o Mac OS, que possibilita a execução de programas em um computador e outros dispositivos, em si, Kernel é o nome do próprio sistema operacional, Linux é apenas o nome dado ao núcleo, Kernel é o software que determina como o como o computador deve exercer sua função, como controlar o uso do processador, da memória, do disco e de dispositivos periféricos. A GNU é um projeto que tinha como objetivo criar um sistema operacional livre, como o Unix, com isso surgiu uma licença de software chamada de GPL, que permite a mudança livre de códigos de um programa. Contudo, basicamente o Linux está em muitos lugares, como no Facebook, ou Netflix, e também em Softwares dominantes como o servidor web Apache, DNS Bind, linguagens de programação PHP, Java, Java scripts, bancos de dados MySQL, postgteSQL e vários outros sob licença GPL, assim executadas em sistemas Linux.

Prós do Linux

Uma das vantagens mais interessantes do uso do Linux está no fato de que há muitas possibilidades em termos de customização. Com o uso de desktops e temas, você pode criar um sistema operacional com uma cara só sua, ou até mesmo reproduzir o aspecto de macOS e Windows.

Usar o Linux envolve mudanças de alguns hábitos e, em virtude das especificidades desses sistemas, vai exigir um pouco de curiosidade, paciência e disposição para pesquisar como resolver alguns tipos de problemas que podem ser de solução trivial em outros sistemas.

Isso significa que usar qualquer Linux acaba permitindo que você aprenda mais sobre o computador, sobre como ele funciona e acaba forçando você a desenvolver habilidades extras, especialmente envolvendo o temido Terminal e as linhas de comando.

o Linux permite que você redefina completamente o comportamento do sistema e tenha um controle bastante preciso a respeito dos seus dados e privacidade, elementos que são vistos por muitos como os pontos mais fracos de plataformas como o Windows. O Linux oferece até mesmo distribuições completamente criadas para garantir a privacidade do usuário.

Contras do Linux

Um problema do Linux está no fato de que sistemas desse tipo falham na tarefa de serem amigáveis a usuários comuns. A dependência excessiva das linhas de comando e Terminal, terminologia e até mesmo a própria comunidade podem representar fatores de grande dificuldade para quem não tem tanta desenvoltura com computadores e resolve experimentar o sistema operacional livre pela primeira vez.

Ao contrário do Windows, que inevitavelmente possui drivers de boa performance para o hardware que você possui, o Linux depende da boa vontade de fabricantes e até de usuários comuns para que componentes tenham suporte via drivers no sistema. Essa realidade é responsável por desempenho inferior, problemas gerais de performance, travamentos e incompatibilidades que podem até mesmo ser incontornáveis em alguns casos.

As ferramentas que você está acostumado a usar no Windows ou macOS inexistem no mundo Linux.
